## Задача

Стрелочная функция — это не просто «короткая запись обычной». У неё другое поведение `this`, нет своего `arguments`, и в некоторых местах она не подходит. На лекции был базовый синтаксис — остальное раскапываешь сама.

```
arrow ≠ просто короткий синтаксис:
  своего this нет — берёт из окружения
  нет arguments
  нельзя как конструктор
ищешь сама: когда arrow удобнее, а когда сломает
```

## Что нужно сделать

В `arrow.js` на примерах показать ключевые отличия arrow от обычной функции: как arrow берёт `this` из окружения (а обычная — определяет свой при вызове); что у arrow нет `arguments`; привести случай, где arrow удобнее (короткий колбэк), и случай, где её использовать НЕ стоит. Выводы — комментариями.

## Критерии приёмки

* Показано отличие в `this` между обычной и arrow-функцией на конкретном примере.
* Показано отсутствие `arguments` у arrow и предложена замена (rest-параметры).
* Приведён случай, где arrow уместна, и случай, где нет; выводы записаны своими словами.
